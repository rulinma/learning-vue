# VUE

## 学习步骤

### 添加脚手架

``` shell
npm install -g @vue/cli
```

### 脚手架创建

``` shell
vue create learning-vue
```

### Project setup

``` shell
yarn install
```

### Compiles and hot-reloads for development

``` shell
yarn serve
```

### Compiles and minifies for production

``` shell
yarn build
```

### Lints and fixes files

``` shell
yarn lint
```
